#vnews
A web app to read news from different sources and categories.

##Getting started

###Prerequisites 
- Npm
- NodeJs

### Installation
- Clone the repository. 
- Run npm install
- Run ng serve
- Open localhost 4200

## Deployment 
- Run ng build --prod
- Copy the contents of the dust folder on the server toot directory. 

## Built with
- Angular-cli 8
